# Valerie Burd's Personal Website

This is a simple static website showcasing my personal resume, contact information, and hobbies.

## Project Structure

- `index.html`: Main resume page with summary, education, work experience, skills, awards, and links to other pages.
- `contact.html`: Contact details including address, phone, and email.
- `hobbies.html`: A list of my hobbies.

## Technologies Used

- HTML5
- Basic CSS (inline styles)

## How to View

Simply open `index.html` in any modern web browser. No server required as it's a static site.

## Navigation

The main page (`index.html`) includes links to the hobbies and contact pages.
